# React_auth

### Server
- MongoDB
- Node.js

### Client
- React.js

## Installation
- Make ```dev.js``` in ```server/config``` dir and paste
```
module.exports = {
    mongoURI:'<Your MongoDB URI>'
}
```

```sh
cd react_auth-master
npm run dev
```
Using 3000 & 5000 port for client, server
